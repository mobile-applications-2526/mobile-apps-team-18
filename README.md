## Get Started

### Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Run the backend server:
   ```bash
   mvn spring-boot:run
   ```

### Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start your Android SDK.
4. Run the Android app:
   ```bash
   npm run android
   ```
